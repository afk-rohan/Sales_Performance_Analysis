# Sales Performance Analysis

A data analytics project that analyzes sales data and presents insights through an interactive **Streamlit dashboard**.
The project focuses on sales trends, KPIs, and visual insights to support business decision-making.

---

## Features

* Sales data cleaning and preprocessing
* KPI calculation (revenue, orders, trends)
* Interactive charts and visualizations
* Streamlit-based dashboard
* Modular and readable Python code

---

## Tech Stack

* Python 3
* Streamlit
* Pandas
* NumPy
* Matplotlib / Seaborn
* SQL (if applicable)

## How to Run Locally

### 1. Clone the repository

```bash
git clone https://github.com/afk-rohan/Sales_Performance_Analysis.git
cd Sales_Performance_Analysis
```

### 2. Create virtual environment

```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the Streamlit app

```bash
streamlit run app_streamlit.py
```
